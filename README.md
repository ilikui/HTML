# HTML基础教程





### HTML 简介

`HTML是用来描述网页的一种语言`

* HTML指的是超文本标记语言（**H**yper **T**ext  **M**arkup **L**anguage）
* HTML不是一种编程语言，而是一种**标记语言**（markup language）
* 标记语言是一套**标记标签**（markup tag）
* HTML使是**标记标签** 来描述网页
